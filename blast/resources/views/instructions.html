This module provides a wrapper around BLAST, allowing users to run/save BLAST searches against custom reference databases. These custom databases allow the user to either create reference sets more tailored to a specific area of research and potentially use non-public sequences. The BLAST module is tightly connected to DISCVR and the SequenceAnalysis module. The latter is used to create and manage all reference sequences. See below for additional instructions on initial configuration and use:

Initial Configuration:

To use the BLAST module, a site admin must install BLAST+ and set several site settings. Site admins can click here to view additional information. Admins can also access this page from the Admin Console under the 'Laboratory Module Admin' link.

Creating Custom Reference Databases:

The SequenceAnalysis module is used to manage all reference sequences. Please refer to that module's documentation for more detail on importing references. Once a reference genome (ie. group of reference sequences) has been created, you can view those libraries by clicking the 'Browse All' link under the 'Sequence' section:



Next, look for the link to view all reference genomes:



If you follow that link, you will see a listing of all available libraries. Check the box next to the library of interest and then use the 'More Actions' button to create a BLAST database from this library. You only need to do this once per library.



Running BLAST:

To run BLAST, go to the front page of your lab's site and look for the 'Run BLAST' link:



This should load a form that looks vaguely like NBCI's public form. From this form, you can enter your query sequence, choose the BLAST algorithm and choose the reference database:



Once you hit the BLAST button, you will be redirected to a page where you can wait the BLAST job's progress and view the results once complete.
